ATIVIDADE 1 – Versionamento de Código com GitHub


Descrição do Projeto

- Este repositório foi criado como parte da Atividade 1 do curso, cujo objetivo é praticar o uso de versionamento utilizando Git e GitHub, garantindo organização, segurança e controle das alterações realizadas durante o desenvolvimento de projetos.

Objetivo da Atividade

- Criar um repositório no GitHub
- Versionar um projeto utilizando Git
- Realizar commits com mensagens claras e descritivas
- Enviar (push) todas as alterações para o repositório remoto
- Entregar evidências do versionamento

Tecnologias Utilizadas

- Git
- GitHub
- GitHub Desktop
- Projeto local da disciplina

Histórico de Commits

Os commits realizados neste repositório seguem boas práticas de versionamento, incluindo:
- Commit inicial – criação da estrutura base
- Adição dos arquivos do projeto
- Atualização do README
